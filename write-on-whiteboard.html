This page will let you use SVG or HTML-building commands at the console and return the documents for copying and pasting into your text editor.

Available functions:
-buildSVG(target) where target is the name (e.g., 'soccer') in the js object Svgs in file napkin-data.js.
-buildHTML(target) does the same thing but wraps the SVG in an HTML shell.
